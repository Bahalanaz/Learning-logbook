# React Fundamentals Practice Project

This repository documents my progress in learning React fundamentals through hands-on practice with components, props, conditional rendering, lists, events, and styling.

---

## 🚀 Project Overview

This project focuses on building reusable React components and combining them into a simple UI system.

Key features implemented:
- Functional components
- Props and default props
- PropTypes validation
- Conditional rendering
- List rendering using `.map()`
- Event handling
- CSS styling and UI enhancements

---

## 🧠 Concepts Learned

### 1. React Components
React is built using reusable functional components.  
Each component represents a part of the UI and returns JSX.

---

### 2. Props (Data Passing)
Props allow data to flow from parent to child components, making components reusable and dynamic.

---

### 3. Default Props
Default props are used when no values are passed to a component, ensuring fallback values are displayed.

---

### 4. PropTypes Validation
PropTypes are used to validate the type of data passed into components and help prevent bugs.

---

### 5. Conditional Rendering
Conditional rendering is used to display different UI based on conditions such as login status.

---

### 6. List Rendering
Arrays are rendered dynamically in React using `.map()`, allowing efficient UI generation from data.

---

### 7. Event Handling
Event handling allows components to respond to user actions such as clicks.

---

### 8. Styling in React
CSS classes were used to style components, including buttons, lists, and message displays.

---

## 🛠️ Features

### 👨‍🎓 Student Component
Displays student information such as name, age, and student status using reusable props.

---

### 👋 UserGreeting Component
Displays a welcome message when logged in and a login prompt when not logged in.

---

### 📋 List Component
Displays dynamic lists of fruits and vegetables with categories and structured layout.

---

### 🔘 Button Component
Implements a clickable button that responds to user interaction.

---

### 🎨 UI Styling
The UI includes:
- Clean component layout
- Hover effects on list items
- Gradient-based message boxes
- Styled buttons with neon-like effects

- Props are used for data flow
